pathfinder_combat
=================

Android Pathfinder Combat
Welcome!

This project is for the Pathfinder role playing game. Easily toggle various buffs and combat modifier to see how they change your character. Automatically see the effects of temporary modifiers to stats, temporary damage increases, critical hits, enlarge person and more.

The objective of this project is create 2 applications, one which serves like a digital character sheet and other which serves to help the master with iniciative tracker, and other things.

How To Debug
=================

In android terminal( eg. terminal emulator ) or with adb, type: logcat PFCombat:V ActivityManager:V ActivityThread:V *:S
